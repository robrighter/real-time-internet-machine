The Real Time Internet Machine

The Real Time Internet Machine provides a quick way to create a real time data feed of any information on your website. This service provides you with 2 things:

  1. A url too which you will POST feed updates
  2. A bit of javascript that to include in your web pages that will pull in the real time feed.
Create a Machine

Post to url:

http://realtimeinternetmachine.com/98u3498rjdskfhsdf3983osidfkjlsfl

Each time you post to this url it will create a new item in the real time feed. You can post items to this url with up to 5 properties. Each property can be 300 characters long.
