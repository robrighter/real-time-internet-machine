Adding items to your Real-Time Machine

Post to url:

http://{{domain}}/insert/{{feedhash}}

Each time you post to this url it will create a new item in the real time feed. You can post items to this url with up to 5 properties. Each property can be 500 characters long.

In addition to the properties of your object, each POST must include an insertkey variable. The insert key for this feed is:

Insert Key:

{{insertkey}}

Adding this Real-Time Machine to a web page

Client Side Javascript:

<script type="text/javascript">document.write(unescape("%3Cscript src='http://{{domain}}/client/{{feedhash}}.js' type='text/javascript'%3E%3C/script%3E"));</script> <script type="text/javascript"> try {
var thertm = new realtimemachine(function(item){
    //Put your code here to insert your item into the DOM
});} catch(err) {}</script>
